A collection of text dumps from Pokémon Champions on the Nintendo Switch.
- __rom-txt__: localized text dumped from the game rom. This can only be updated when the game is patched (i.e. version is updated).
- __masterdata__: tables that can be updated without a game update. This is usually updated when the game asks to download additional data on boot.

Git history can be viewed to see changes from prior versions.
